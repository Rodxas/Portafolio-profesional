📝 Forms en Angular

Angular soporta dos tipos de formularios: Template-driven y Reactive

1. Template-driven (ngModel)

Ideal para formularios simples. La lógica está en el template.

@if (submittedContacts.length > 0) {

Mensajes enviados ({{ submittedContacts.length }}):

@for (c of submittedContacts; track $index) {
{{ c.nombre }} - {{ c.email }}
}
}

2. Reactive Forms (FormGroup)

Ideal para formularios complejos. La lógica está en el componente (TypeScript).

@if (isInvalid('nombre')) { {{ getError('nombre') }} }
@if (isInvalid('email')) { {{ getError('email') }} }
@if (isInvalid('password')) { {{ getError('password') }} }

📊 Comparación

Característica Template-driven Reactive
Complejidad Baja Alta
Validación En template En TypeScript
Testing Más difícil Más fácil
Casos de uso Forms simples Forms dinámicos/complejos
