✨ Directives en Angular

Las directivas manipulan el DOM y controlan la estructura del template

1. *ngFor - Renderizar listas

Iterando sobre un array de frutas:

    @for (fruta of frutas; track fruta) {
  • {{ fruta }}
    • }

2. *ngIf - Condicionales

@if (isLoggedIn) {
✅ Usuario autenticado | Rol: {{ userRole }}
} @else {
🔒 Por favor, inicia sesión para continuar
}

3. *ngSwitch - Múltiples casos

@switch (opcionActual) { @case ('1') {

📦 Contenido de la Opción 1

} @case ('2') {

⚙️ Contenido de la Opción 2

} @case ('3') {

🎯 Contenido de la Opción 3

} }
